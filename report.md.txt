# Self-Pruning Neural Network — Report

## Why does L1 on Sigmoid gates encourage sparsity?
The sigmoid function squashes gate scores to values between 0 and 1.
The L1 penalty (sum of all gate values) directly penalizes the network
for keeping gates "open". Since reducing any gate value toward 0 reduces
the loss, the optimizer is incentivized to zero out gates for unimportant
weights, creating a sparse network.
